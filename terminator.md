# terminator

## install 
```
sudo apt install terminator 
```

## shortcuts
```
Ctrl + Shift + e   :  수직 분할 
Ctrl + Shift + o   :  수평 분할
Ctrl + Shift + p   :  이전 창 활성화 
Ctrl + Shift + n   :  다음 창 활성화 
Ctrl + Shift + w   :  현재 창 닫기 
Ctrl + Shift + q   :  terminator exit
F11                :  전체 화면 

Ctrl + Shift + 방향키 : Split 창 크기 조절 
Ctrl + Tab            : Split 창 간 이동 

Ctrl + Shift + t     : Tab 생성 
Ctrl + Page Up/Down  : Tab 이동 
```
