# vim shortcuts 


## insert 관련 
|   |                                |
|---|--------------------------------|
| i | 커서 앞에 insert               |
| I | 줄 맨 앞에 insert              |
| a | 커서 다음에 insert             |
| A | 줄 맨뒤에 insert               |
| o | 커서 아래에 한줄 띄우고 insert |
| O | 커서 위에 한줄 띄우고 insert   |

## 위치 관련 

|        |           |
|--------|-----------|
| Ctrl+o | 이전 위치 |
| Ctrl+i | 앞전 위치 |
