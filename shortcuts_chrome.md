# chrome shortcuts 

## 창 관련 
|                   |                  |
|-------------------|------------------|
| 새 창             | Ctrl + n         |
| 새 탭             | Ctrl + t         |
| 시크릿 창 열기    | Ctrl + Shift + n |
| 닫은 창 다시 열기 | Ctrl + Shift + t |
| 뒤로 가기         | Alt + Left       |
| 앞으로 가기       | Alt + Right      |
|                   |                  |
|                   |                  |
|                   |                  |
